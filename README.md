# Scrapybara Cookbook 🦝🍳

Welcome to the Scrapybara Cookbook! This collection of Google Colab notebooks will help you master the art of web scraping and automation using Scrapybara. Whether you're a beginner or an experienced developer, these recipes will guide you through various scraping scenarios and techniques.

## 🌟 Getting Started

To use these recipes, you'll need:

1. A Scrapybara API key (Get one [here](https://scrapybara.com))
2. Access to Google Colab

Each notebook in this cookbook is designed to be self-contained and easy to follow. Simply open the notebook in Google Colab, add your API key, and you're ready to start scraping!

## 📚 Recipes

1. **[Scraping Fashion Websites](https://colab.research.google.com/drive/1efD3dmEmK3Oz4qFk9uU1nCU7_lB-T1eP?usp=sharing)**

   - Learn how to scrape product information from fashion websites. This notebok shows how to execute existing scripts in a composable way with for loops.
